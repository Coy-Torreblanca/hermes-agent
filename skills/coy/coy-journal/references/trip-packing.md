# Trip Preparation Pattern

When Coy has an upcoming trip, maintain a running packing list. He'll add items conversationally — capture each one.

## Setup

1. Create a gbrain page: `slug: <trip>-packing-list`, type: `checklist`
2. Include trip dates and itinerary in the page header
3. Use org-style checkboxes: `- [ ] item`

## During the trip lead-up

- Coy will mention items in passing ("first item is my dry scalp shampoo")
- Add each immediately to the gbrain page
- Don't wait for a formal "packing session" — capture as he thinks of things
- Some items are experimental / non-critical — note these (e.g., "heavy — experimental, not critical")
- When Coy explicitly says he's NOT bringing something (e.g., "No Switch in VR headset"), add a "Did NOT Bring" section. This prevents later confusion where Coy asks "did I bring X?" — the answer is captured.

## Example

```
slug: mexico-packing-list
type: checklist
---
Trip dates: May 8–29, 2026
Destin → San Diego → Mexico (PVR) → San Diego → Destin

## Packing List
- [ ] Dry scalp shampoo
- [ ] ...

## Did NOT Bring
- Nintendo Switch
- Apple VR headset
```
